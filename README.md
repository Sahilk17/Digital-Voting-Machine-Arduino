Digital Voting Machine using Arduino

Project Description

This project implements a Digital Voting Machine using Arduino Uno. Three push buttons are used for voting candidates A, B, and C. LEDs indicate successful vote registration. A result button displays vote counts and declares the winner.

Components Used

- Arduino Uno
- 4 Push Buttons
- 3 LEDs
- 220 Ohm Resistors
- Jumper Wires

Features

- Vote counting
- LED indication
- Result declaration
- Debouncing logic
- Secure vote storage during execution

Working

Each candidate has a dedicated button. When a voter presses a button, the vote count increases and the corresponding LED glows briefly. Pressing the result button displays total votes and the winner.
